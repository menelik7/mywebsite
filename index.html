A reverse-phone lookup application that is quite user-friendly. There are no hoops to jump through; no unnecessary steps to complete. The user simply enters the phone number he/she wishes to lookup, and voila! Information specific to the user is dynamically displayed, as well as the option to lock-in on the associated address with the help of Google Maps API. The reverse phone is made possible by Whitepages API.

I worked on a team to develop this app, and I was responsible for the logic and database (Javascript & Firebase), AJAX method for API queries, as well as Firebase Authentication.
